# SniptCart – Issue Tracker

**Official public issue tracker for SniptCart Platform, SDKs, and Admin Dashboard**

SniptCart is a modern, developer-first commerce platform designed for rapid integration, flexible customization, and scalable digital commerce.
This repository serves as the **central place to report bugs, request features, and track improvements** across:

* **SniptCart Platform APIs**
* **SniptCart JavaScript / TypeScript SDK**
* **Mobile / Server SDKs**
* **SniptCart Admin Dashboard**

If you're building with SniptCart or integrating commerce into your product, this is the right place to file issues.

---

## What is SniptCart?

SniptCart is a **lightweight, API-first e-commerce engine** built for developers.
It powers:

* Online stores
* Checkout experiences
* Restaurant & food ordering flows
* Subscription billing
* Digital product sales
* Multi-tenant commerce systems
* Custom front-end commerce experiences (Next.js, Vue, React, etc.)

Learn more at: **[https://sniptcart.com](https://sniptcart.com)**
